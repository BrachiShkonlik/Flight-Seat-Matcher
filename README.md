# Flight Seat Matcher

## Overview
Flight Seat Matcher is a web application designed to enhance the flight experience by matching passengers to seats based on their age and hobbies. This project uses MongoDB for database management, C# for backend processing, and React for the frontend interface.

## Features
- **Profile Creation:** Allows passengers to create profiles with their age and list of hobbies.
- **Seat Matching Algorithm:** Matches passengers to seats that align with passengers of similar age and hobbies.
- **Interactive Seat Map:** Visualizes the seating arrangement on the flight and allows passengers to view their seat assignment.
- **Admin Panel:** Enables airline administrators to view and manage seat assignments and passenger profiles.

## Technologies Used
- **Frontend:** React, HTML, CSS
- **Backend:** C# (.NET Core)
- **Database:** MongoDB
- **Other Libraries and Tools:** Axios for API requests, Mongoose for MongoDB object modeling
